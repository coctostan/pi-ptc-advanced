## How to use this archive
Use these artifacts only when you need one of the following:
- recover the old upstream submission rationale
- rebuild a clean review branch manually from the archived instructions
- inspect why a file was treated as upstream-worthy, local-only, or follow-up
- reference the old PR narrative without re-deriving it from milestone history

If you are just maintaining the fork locally, stop here and use the personal maintenance runbook instead.

## Why this remains archived instead of deleted
The Milestone 7 material still contains useful historical decisions and executable reference notes, especially around scope boundaries and restore-based review branch construction.

Deleting it would lose context. Keeping it in place without an archive guide would make the repo harder to navigate.

This archive page keeps the history discoverable while making the current workflow unambiguous.
